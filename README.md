## Featured Projects

### 🔧 [quantprobe](https://github.com/awsmak/quantprobe) *(in development)*
A pip-installable tool for diagnosing why quantized ONNX vision models lose 
accuracy. Per-layer sensitivity analysis, visual HTML reports, mixed-precision 
recommendations. Vendor-neutral.  
**Stack**: Python, ONNX Runtime, NumPy, Plotly

### 🚗 [FusionTrack3D](https://github.com/awsmak/fusiontrack3d) *(in development)*
Multi-modal 3D object detection and tracking that fuses LiDAR point clouds with 
camera imagery — relevant to autonomous driving perception stacks.  
**Stack**: PyTorch, point cloud processing, sensor fusion

### 🌱 [Optical Plant Detection](https://github.com/awsmak/Optical-Plant-Identification-for-Farming-Vehicles-)
Real-time weed detection (Rumex Acetosa) on Raspberry Pi using custom YOLOv3 
and EfficientDet Lite — a complete edge AI deployment from training to inference.  
**Stack**: TensorFlow, EfficientDet, ARM deployment

### 🏥 [Retina Blood Vessel Segmentation](https://github.com/awsmak/retina_blood_vessel_segmentation)
UNet-based semantic segmentation for medical imaging.  
**Stack**: PyTorch, UNet architecture
